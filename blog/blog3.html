This week, we learned about aboslute positioning. When an element is positioned absoutely, it anchors itself to the nearest ancestor that has non-static positioning. Since the default positioning for elements is static, this means that we have to set the parent element that we want the child to anchor to with relative positioning instead. Another cool thing we learned this week was how to mimic powerpoint slides and their transitions in lab 7/8.My group struggled with the transitions but after seeing the solution, we saw that we overthought it and made it much more complicated than needed. Part of the error was because we were trying to use transition instead of transform.

I was interested in finding other ways to use absolute positioning on a web page, so I looked up other examples and came across this YouTube video tutorial on practical examples for this positioning. Not only was it very helpful with making me understand positioning better, but I got to learn some new things about CSS such as calc and rem. I have a better understanding of how to use rem and em now after watching his tutorial.
Some example CSS code used in the video that I slightly altered and recreated while watching the video is shown below and also appears embedded on this blog post: 

                
                    .card {
                        position: relative;
                        width: 40%;
                        margin: 10px;
                        box-shadow: 0 0 25px 0 rgba(0, 0, 0, .35);
                    }
                    
                    .card::before,
                    .card::after {
                        content: '';
                        position: absolute;
                        border: 1px solid white;
                    }
                    
                    .card::before {
                        top: 1.5rem;
                        bottom: 1.5rem;
                        right: 1.5rem;
                    }
                    
                    .card::after {
                        bottom: 1.5rem;
                        left: 1.5rem;
                        right: 1.5rem;
                    }
                    
                    .card-img {
                        position: absolute;
                        width: 100%;
                        height: 100%;
                        object-fit: cover;
                        /* pushes it behind */
                        z-index: -1;
                    }
                    
                    .card-body {
                        background: rgba(0, 0, 0, .5);
                        color: white;
                        padding: 3rem;
                    }
                    
                    .card-title,
                    .card-text {
                        margin: 0;
                    
                    }
                    
                    .card-title {
                        font-size: 3rem;
                        line-height: 1;
                        margin-bottom: .5em;
                        position: relative;
                        /* pulling card title forward and on its own layer */
                        z-index: 1;
                        color: black;
                    }
                    
                    .card-title::before {
                        content: '';
                        position: absolute;
                        background: yellow;
                        /* 100% of the card title width plus an additional 5rem to compensate for the excess 5rem from left */
                        width: calc(100% + 5rem);
                        top: -0.1em;
                        bottom: -0.15em;
                        left: -5rem;
                        z-index: -1;
                        /* only want the shadow on the right side */
                        box-shadow: inset -15px 0 10px -10px rgba(0, 0, 0, .5);
                    }
                    
                    .card-title::after {
                        content: '';
                        position: absolute;
                        top: -0.1em;
                        bottom: -0.15em;
                        left: calc(100% + 3rem);
                        width: 2rem;
                        background: yellow;
                        box-shadow: inset 12px 0 10px -10px rgba(0, 0, 0, .5);
                    }
                    
                    .card-text {
                        text-align: right;
                    }
